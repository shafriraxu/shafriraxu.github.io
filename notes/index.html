NOTES

  • 相對路徑

    / → root
    ./ → 目前的path
    舉例如：kissr中有Ex02.html和images文件夾，images文件夾中有01.jpg，02.jpg等等，此時用相對路徑比較方便。

  • CSS樣式的定義

    CSS要另存在一個資料夾中，html中用link鏈接。指令自己查哦～
    基本格式：
    name{
    屬性：值；
    屬性：值；
    }
    關於name:
    （1）.name是英文，默認是tag name
    （2）#name是id name，在tag後面加上id='name'，是唯一的。
    （3）.name是class name，用法為在tag中加上class='name'，允許多個。
    （4）a.w3-btn 表示只有a的tag才能用。

  • 架構樹

    這個說來話長，看照片吧。

    there

  • CSS的關係

    （1）父子關係
    .ul>li>span{}
    （2）祖先關係
    .ul li span{}
    （中間可以有多個標籤，只是包含關係。）
    （3）ul,li,span{}
    （三個tag用同一個樣式）

  • 表格

    table由thead和tbody組成，tbody中有tr，tr中有td。
    HTML 表单中有两种类型的单元格：
    （1）表头单元格 - 包含表头信息（由 th 元素创建）
    （2）标准单元格 - 包含数据（由 td 元素创建）
    th 元素内部的文本通常会呈现为居中的粗体文本，而 td 元素内的文本通常是左对齐的普通文本。

  • 一些小提示

    （1）外連圖片加載速度較慢。
    （2）chrome瀏覽器按F12或者右鍵可以審查元素；
    （3）將"meta charset='utf-8'"用尖括號括起來後放在head中，就會鎖定用utf-8啦。
    （4）最好用css將標籤下的東西統一修改，自定義名稱前面要加“.”。
    （5）JavaScript的鏈接要放在body的最後一行喔。
    “container-fluid”是全版面的row。
